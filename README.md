# 🎬 Clipmaster Update

Ứng dụng hỗ trợ xử lý video tự động: ghép chữ, xử lý hàng loạt, tối ưu nội dung và workflow đăng tải.

---

## 🚀 Tính năng chính

* ✍️ **Thêm chữ vào video**

  * Hỗ trợ nhiều dòng chữ
  * Tùy chỉnh vị trí (%X, %Y)
  * Font, màu sắc, kích thước linh hoạt

* 🖼️ **Thêm ảnh vào video**

  * Chèn ảnh overlay
  * Tùy chỉnh vị trí và kích thước
  * Phù hợp làm watermark hoặc nội dung quảng cáo

* 🔤 **Đổi tên file hàng loạt**

  * Rename file theo quy tắc
  * Tự động hóa xử lý thư mục lớn

* 🎯 **Ghép chữ + ảnh tự động**

  * Kết hợp nhiều layer
  * Xử lý hàng loạt nhiều video

* ⚡ **Tối ưu hiệu suất**

  * Giảm thời gian render
  * Tối ưu hiệu suất
